在今年南半球墨尔本的夜晚，人们正在抬头仰望一场特别的「有机烟花」，完成关于灯火阑珊的另一种想象。在正视各种污染与浪费的设计探索中，来自荷兰的艺术家与创新者 Daan Roosegaarde 为传统的烟花、气球、纸屑、孔明灯等庆祝方式提供了一种可持续的替代方案 SPARK。

没有火，不产生烟和噪音，由数十万个可反射和吸收光线的可生物降解的气泡组成，一旦释放，就可以乘风在空中漂浮、游动、上升，就像大气中的生物发光，被凉爽的夜风和人类的眼睛捕捉到。最终 SPARK 可以上升至 50 米，方圆几公里都可以看到这些美丽的“光火花”。

本周讨论：凯文·凯利的73岁感悟

凯文·凯利（Kevin Kelly），常被称为“KK”。他是《连线》杂志（Wired Magazine）第一任主编；曾担任《全球评论》（Whole Earth Review，乔布斯最喜欢的杂志）主编和出版人；同时是位作家，著有《失控》、《必然》等书。

在每年生日之时，KK 都会撰文分享他的生活经验和感悟。我们曾经在 62 期的时候分享了他在 70 岁时的感悟，他也把每一年的这些感悟在 2023 年总结成了一本书：

今年他快要 73 岁了，在他的博客分享了 101 条在书中从未出现过的全新建议。这些建议仍然值得学习，精选翻译一些，分享给大家。

01.

一个人是无法真正成为自己的。成为独一无二的自己不是一个人的事。矛盾的是，你需要世界上的每一个人帮助你成为独一无二的人。

02.

要想成功，最可靠的方法就是创造自己对成功的定义。先把箭射出去，然后在箭的落点周围画一个靶心。你就是赢家！

03.

当你发现自己拖延时，不要抗拒。相反，靠进去。拖延 100%。尝试在 5 分钟内什么都不做。让它成为你的工作。你会失败的。5 分钟后，您将准备好并渴望工作。

04.

如果你想知道外科医生有多好，不要问其他医生。问护士。

05.

观点鲜明，表述清晰，但又松弛有度，这就是知识分子生活的秘诀。经常问问自己：怎样才能改变我的想法？

06.

你没有义务喜欢所有人，你可以自由地强烈地不喜欢一个人。但你欠每个人——即使是那些你不喜欢的人——基本的尊重。

07.

忘掉试图决定自己人生命运的想法吧。那太宏大了。相反，只要想清楚未来两年应该做什么就可以了。

08.

无论何时拥抱他人，都要最后一个松手。

09.

不要把好东西（高档酒或瓷器）留到永远不会出现的罕见场合，而是要随时使用它们。

10.

最好的园艺建议：找到你能种好的东西，并种很多很多。

11.

千万不要犹豫投资自己–花钱去上一堂课、学习一门新技能。这些微薄的支出会带来巨大的回报。

12.

在父母健在时采访他们。一边录音，一边不断提问。你会学到令人惊叹的东西。或者雇人将他们的故事制作成口述历史、纪录片或书籍。这对他们和您的家庭都将是一份巨大的礼物。

在购买任何实物（纪念品、家具、书籍、工具、鞋子、设备）时，问问自己：这东西要放在哪里？除非它有地方住，否则就不要买。为了让其他东西进来，有些东西可能需要离开。

14.

多任务处理是个神话。不要在走路、跑步、骑车或开车时发短信。只要你停下来一分钟，没人会错过你。

15.

在你的生活中，至少应该有一件事是你喜欢的，尽管你并不擅长。这就是你的游戏时间，它能让你永葆青春。永远不要为此道歉。

16.

在一个新想法从你脑海中消失之前，你有 5 分钟的时间采取行动。

17.

不要害怕失败。但要害怕平凡。

18.

20.

当你尝试新事物时，不要把它看成是成功与失败的问题，而是成功与学会成功的问题。

21.

24.

1、世界上最高的天文台（英文）

经过 26 年的规划和建设，东京大学阿塔卡马天文台（TAO）于周二正式开放。这架 6.5 米高的光学红外 TAO 望远镜位于智利阿塔卡马沙漠的 Chajnantor 山上，高 18,500 英尺，现在是世界上最高的望远镜。

如此之高意味着空气中的水分要少得多;TAO 可以观测到“几乎整个近红外波长范围”，包括中红外波长。Phys.org 指出，没有其他地球望远镜可以做到这一点。东京大学写道，这种地面天文台能够拍摄更高分辨率的太空照片，这要归功于它们的光圈比天基天文台更大。根据东京大学的公告，从 2025 年开始，该望远镜将用于了解“星系的诞生和行星的起源”。

然而，TAO 坐在如此极端的高度上的好处是有代价的，因为人类不适合在这么高的地方生活。

正如 Phys.org 所指出的，即使是在里面工作的研究人员也需要采取预防措施，以便在面对高原反应时保持健康。该团队计划最终从较低的基地设施远程操作望远镜，以避免此类问题。

2、在火星上看夜空（英文）

在火星上进行夜晚观星是什么样的体验？首先，火星上很少有多云的夜晚，这使得它在很多方面成为业余天文学的理想地点。火星的夜晚通常和地球上清晰的沙漠天空一样清澈。你会看到两颗卫星——火卫一（Phobos）和火卫二（Deimos），而不是一个。

从火星基地黄昏时出来，你会在多岩石的地貌上空的火星夜空中看到美丽的日落。

由于多尘的大气，你会看到紫色和蓝色的日落，太阳在落到地平线上的轮廓分明的火山后面之前，会缩小成一个缩小的蓝色硬币。

好奇号漫游者拍摄的照片表明，太阳落山后，你有很好的机会看到银色蓝色的夜光云在暮色中闪耀，甚至可能是一场令人惊叹的展示，让我们自己夏日天空中的任何景象都相形见绌。根据一年中的时间，你也可能看到一颗“晚星”在薰衣草色的暮色中闪耀。不是金星，而是地球，一颗明亮的银蓝色火花，在其最佳状态下会比-2.5等更亮。

如果地球显示出满月或凸月相，通过你的望远镜，你将看到熟悉的绿色大陆和蓝色海洋在白天的一面，以及城市灯光在夜晚的一面闪烁。

3、刺绣背面，是健康崩塌

在快节奏的都市中，亚健康不是一个新鲜的词汇。许多人多多少少都存在亚健康的状态，但由于其危害尚不明显，尚未引起足够的重视。最近，一位日本大学生馬杉帆々香采用了一种新颖且有视觉冲击力的表现方法，让人对亚健康的危害一目了然。

她的作品主题是为兴和制药的产品“キューピーコーワヒーリング锭”做推广。这种药物可以提高睡眠质量、缓解疲劳症状，改善都市人群的亚健康状态。为引起人们对身体状况的重视，馬杉帆々香 用一种乍一看和健康压根毫无关系的事物来承载创意——刺绣。

她将手工刺绣拆解为正反两面，将其并列展示。刺绣正面寓意人们对外展示出的面貌，反面则代表亚健康状态真实的模样。即便一个人表面看起来充满活力，但实际上疲劳、失眠等等问题已经将ta的身体和精神折磨地如刺绣另一面——在混乱的线条中失去了原有的平衡感。

对刺绣双面性的巧妙利用，让这幅作品也获得了专业人士的青睐。4 月 3 日，馬杉帆々香凭借此作品，获得第 91 届日本每日广告设计奖“广告主命题”类优秀奖作品。

1、腐烂经济（英文）

这篇文章讨论了现代商业资本部署中一个严重的问题，即公共和私人投资者以及市场本身已经与“良好”业务的真正概念完全脱节，他们只关注一个指标——增长。这里的“增长”并不是指变得更“大”或“更好”，而只是指“更多”。

这意味着公司正在产生更多的收入、更高的估值、获得更多的市场份额，并找到更多产生这些东西的方法。

在公共市场上，这意味着像谷歌、Meta 和微软这样的公司因其无焦点、资本密集型的业务而受到奖励，当形势变得艰难时，这些公司需要大规模裁员，因为市场喜欢他们找到了节省资金的方法。他们不会因为糟糕的规划、停滞不前的产品、人力资本管理不善或普遍缺乏真正创新而受到惩罚，因为数字一直在上升。

比如 Meta（Facebook 的母公司）在构建一个永远不会存在的元宇宙上烧钱后，股价受到了打击。然而，一旦扎克伯格解雇了 11,000人，并声称 2023 年将是“效率年”，尽管 Facebook 活跃用户增长下降，他们在同一个元宇宙部门损失了 137 亿美元，市场还是以双位数增长回应了 Meta 的股价。

文章还讨论了谷歌的类似但略有不同的故事，谷歌的核心产品——搜索——已经从一个寻找信息的地方变成了一个越来越被操纵的 SEO 优化垃圾的迷宫。

文章最后指出，风险投资和公共市场实际上并不奖励或尊重“良好”的商业或“良好”的CEO——他们奖励那些能够推动提高资产价值的增长的人。文章呼吁，我们需要在评估企业的方式上进行重大转变，否则我们将继续经历对人力资本的狂饮和净化的惩罚性周期。

2、比免费更好（英文）

这篇文章由凯文·凯利（Kevin Kelly）撰写，题为《比免费更好》（Better Than Free），探讨了在互联网时代，当复制品（copies）变得极其丰富甚至免费时，企业和创作者如何创造价值并从中获益。文章指出，互联网本质上是一个复制机器，它不断地复制我们在其上进行的每一个动作、每一个字符、每一个思想。这种复制是数字经济的基础，但也对传统的基于销售珍贵复制品的商业模式构成了挑战。

凯利提出了一个核心问题：如果复制品是免费的，我们如何通过它们赚钱？ 他的回答是，当复制品变得极其丰富时，它们变得无价值，而不能被复制的东西变得稀缺和有价值。因此，我们需要销售不能被复制的东西。

文章列出了八种不能被复制的价值，凯利称之为“生成性”（generative）价值，因为它们必须被生成、成长、培养和呵护。这些生成性价值包括：

  1. 即时性（Immediacy）：及时获取新内容的能力是有价值的。
  2. 个性化（Personalization）：定制内容以满足个人特定需求的服务。
  3. 解释性（Interpretation）：为免费软件提供的支持和指导。
  4. 真实性（Authenticity）：确保内容是真实和可靠的。
  5. 可访问性（Accessibility）：通过订阅服务方便地获取内容。
  6. 具体化（Embodiment）：将数字内容具体化，如通过高质量的屏幕或3D显示。
  7. 赞助（Patronage）：粉丝愿意为支持他们喜欢的创作者而付费。
  8. 可发现性（Findability）：在大量免费作品中，帮助用户发现和找到作品的能力。

凯利认为，这些生成性价值需要一套新的技能集。在免费复制品的世界中取得成功，不是通过分销技能，也不是通过知识产权和版权的法律技能，更不是通过囤积和稀缺性的技能。相反，这些新的生成性价值要求我们理解丰富如何孕育共享心态，慷慨如何成为商业模式，以及培养和呵护不能通过点击鼠标复制的特质变得至关重要。

文章最后指出，在这个网络化经济中，金钱并不跟随复制品的路径，而是跟随注意力的路径，而注意力有自己的回路。凯利还提到，尽管广告被广泛认为是解决免费的悖论的方案，但它只是注意力所采取的路径之一，在长期内，广告只会是出售免费内容的新方法中的一部分。

3、工程团队从自行车运动中吸取的教训（英文）

本文通过自行车运动的比喻，深入探讨了软件开发团队中协作的重要性。作者指出，像自行车队一样，软件团队在合作时能够比单打独斗更快地实现目标。文章强调了团队合作的空气动力学优势，即通过集体努力可以克服环境挑战，实现更高效的目标达成。同时，作者也提到了个人工作的优势，如在软件团队中，个别成员的专长可以帮助团队抓住机会，解决突发问题。

整体而言，文章倡导在软件开发中灵活运用团队和个人的优势，以适应不断变化的市场和技术环境。

4、 《代码的本质》（第二版）（英文）

这是一本印刷书记的在线版本，探索了通过创意编码框架 Processing 在软件中模拟自然界中不可预测的进化和突现特性。作者提到，自他大约十年前自费出版以来，技术和创意媒体世界已经发生了巨大变化，因此他现在推出了一个围绕 JavaScript 和 p5.js 库构建的新版本（因为它们适合初学者，并且所有代码都可以在浏览器中运行。）。

作者强调了这本书不是严格的科学书籍，也不是艺术或设计书籍，而是一本传统的编程教科书，重点在于面向对象编程。

这里不仅有作者的 Youtube 频道和在线 Coding 平台，提供了与书中内容相关的视频教程。还有一个可能的教学大纲，将书中的内容设计为适合 14 周的课程，并鼓励读者分享他们的工作和反馈。他将这本书视为一个活文档，并鼓励社区参与和贡献。

总之是一个质量相当在线的资料。

5、用户是独立的（英文）

我们要么期望人们超级自给自足，要么期望复杂的事情变得超级容易。

在这篇文章中，作者深入探讨了用户界面设计中的一个核心概念——直观性。她指出，尽管我们期望用户界面直观易用，但现实中几乎没有什么是真正直观的。她通过日常物品的例子，如电源按钮和电钻，说明了即使是看似简单的设备也需要一定的学习和指导。

作者强调，计算机和其他复杂设备的使用同样需要社区支持、良好的文档和可访问的学习资源。她认为，设计界面时应考虑到这些问题，并对那些在使用过程中遇到困难的人表示同情。

这是一篇关于最新的 CSS 新增功能列表的详细说明，提供了基本的使用示例。如果你想了解最新的 CSS 技术和实践，这篇文章是一个很好的资源。

7、在你到达那里之前，你可能不知道你要去哪里（英文）

这篇文章通过阿基米德确认王冠是否是纯金的故事，探讨了直觉在创意过程中的重要性。作者强调，无论是科学还是商业营销，直觉和逻辑都是创新过程中不可或缺的两个阶段。直觉可以引导我们发现新的想法，而逻辑则用于验证和证明这些想法。在品牌策略中，直觉的作用尤为关键，它能帮助我们捕捉品牌的本质，即使这个过程看起来不够精确或专业。

文章最后提出，我们应该坦诚地承认直觉在我们所有真正有创意和价值的工作中所占的份额，并从他人的诚实中获取勇气。

8、看完大模型，我决定改变自己的大脑

文章讲述了作者在学习了大型语言模型（LLM）的工作原理后，决定改变自己的思维模式，向大模型学习。文章以 GPT-3 为例，介绍了大模型如何通过 Transformer 结构和注意力机制来处理语言信息。

作者强调了大模型的特征向量、融会贯通的能力、理解语境和语义的方法、自我修炼的过程，以及举重若轻的能力。文章指出，大模型的智能虽然不同于人类，但其结构和训练过程非常值得人类学习。

1、又臭又腥的石楠花，为什么满大街都种 | 答案如下

石楠花因其含有“1-吡咯啉”（1-pyrroline）而散发出与人类精液相似的气味，这是其腥臭味的来源。尽管石楠花的气味在网络上引起热议和调侃，但它在城市绿化中的应用却非常普遍。石楠花的提取物中还含有苯甲醛和苯乙醇，这些物质的相互作用构成了石楠花复杂的气味。

尽管气味难闻，石楠树因其在景观效果和成本上的高“性价比”而受到园林部门的青睐。课代表总结：

  1. 观赏性高：石楠树，尤其是红叶石楠，因其春季新叶鲜红夺目，具有很高的观赏价值，被称为“红叶绿篱之王”，在城市绿化中能够提供良好的视觉效果。
  2. 经济实惠：石楠树的种植成本低廉，得益于其扦插技术和组培技术的成熟，使得种苗生产成本大幅降低。红叶石楠的价格远低于其他树种，使其成为城市绿化经济上的首选。
  3. 养护成本低：石楠树具有较强的抗旱性和抗寒性，对不利环境的抵抗能力强，因此在日常养护上较为省心，减少了水、肥料、药剂、机械的使用频率，从而降低了长期的养护成本。
  4. 符合绿化原则：石楠作为土生土长的本地植物，符合植物景观优化原则中对乡土性和观赏性树种的保留和推广，这也是其在城市绿化中得到广泛应用的原因之一。
  5. 政策支持：国务院办公厅发布的《关于科学绿化的指导意见》提倡节俭务实推进城乡绿化，推广抗逆性强、养护成本低的地被植物，石楠树的特性符合这一政策导向。
  6. 适应性强：石楠树能够在多种气候条件下生长，适应性强，这使得它成为城市绿化中一个稳定和可靠的选择。

当然了，忍无可忍的你也别太难过，石楠的整个花期也只有 7-10 天，受不了的小伙伴可以避开或者带上口罩，接着就可以尽情享受石楠的好处了。

2、为什么你能感觉到背后有人盯着你 | 答案如下

课代表总结：

  1. 心理作用：人们感觉到背后有人注视自己可能只是一种心理作用，而非实际的感知。这种感觉可能是对环境中微妙线索的无意识反应，如微小的气味、空气流动或温度变化。、
  2. 生理反应：超心理学家通过测量皮肤电反应等生理指标来研究“被凝视感”，认为即使参与者没有意识到，他们的生理反应可能揭示了对他人目光的某种感知。
  3. 环境和刺激性因素：2023 年的虚拟现实研究显示，通过精心设计的环境因素（如黑暗、人物出现、红色调色板和悬疑音乐）和刺激性因素（如视觉和听觉的细微变化、颈后的触觉），可以成功诱发“被凝视感”。
  4. 焦虑和紧张：焦虑或紧张情绪可能会放大人们对于被注视的感觉，提高警惕性，导致人们误将主观体验当作真实事件。
  5. 自我实现的预言：人们可能因为怀疑背后有人注视而回头，这一动作反而吸引了后面人的注意，从而验证了被注视的感觉。
  6. 盲视现象：尽管“被凝视感”可能是错觉，但人类的大脑确实具有无意识的视觉处理能力，如盲视现象所示，这表明大脑能够处理一些我们并未意识到的视觉信息。
