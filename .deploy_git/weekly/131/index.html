本周讨论：取舍才能拥有更好的人生

（本期来源：《选 3 哲学》：取舍才能拥有更好的人生）

今天想要分享的这一本书是《选3哲学》，这一本书是关于〝人生选择〞的智慧，我们不能事事都要求完美，必须有些取舍，才能获得更幸福完满的人生。 在这一本书中，〝取舍〞与〝专注〞才是更重要的，我们不要贪求更多，但我们应该〝专注〞在更重要的事情上，在一天结束前，你知道自己不只做到你订下的 3 项目标，而且每件事都做得很好，这种感觉不是很棒吗？

而这本书作者就是大名鼎鼎 Facebook 创办人的姐姐兰蒂. 祖克柏 （Randi Zuckerberg ），作者为什么是提出〝选 3 哲学〞，是因为她常常被问到一个问题：「兰蒂，你是一名妈妈，而且还有一份事业。 你如何在两者间取得平衡？」 在这个问题的意识下，作者意识到她自己〝也办不到〞取得平衡点。

作者回答说：

为了能够让自己准备好、能够成功，我知道在现实状况下，我一天只能做好 3 件事。 所以每天我早上起来，我会想着，工作、睡眠、家庭、朋友、运动，选择 3 项，明天我可以选不同的 3 项，再一天也可以选不同的 3 项。 但是今天，我就是只能选3项。 只要长期下来每样都有选到，就可以兼顾所有不能同时兼顾的，一举解决创业者的两难。

不管你是靠什么维生，不管你住在哪里，不管你担了什么责任，没人做得到兼顾全部而丝毫不牺牲，或是没少掉一丁点的专注力及精力。 后来我就不说这是「创业者的两难」，我把它改个名称，叫做「选 3 哲学」，这样涵盖得广一点，而且还能即知即行。

选 3 哲学

「无法兼顾」是负面词，但我却认为它是成功与快乐的关键。

这本书开宗明义就直接挑明的说：「认清事实吧！ 你就是不可能达到完美平衡」;

人的时间是有限的，不可能全部合理分配时间在每一个方面，人生不可能事事完美，正因为如此，所以〝选择〞便如此的重要。 我不否认偶尔一两天你可以 5 项都做到，但是长期来说，这样真的不能永续。 如果你试图把 5 件事都做好做满，那就是走向耗竭。 你没有办法维持在高功能状态把 5 项都做到最好。 家庭、朋友、工作、睡眠、运动，每天都碰一下，当然做得到。 但是，样样通就样样松，就算只有一天达到 5 项满贯，这表示每一项都不够有深度。

既然我们无法在〝短期〞内获得完美平衡的人生，那么我们否有可能在〝长期〞之间或许相对平衡的人生选择呢？ 我们向来接收到的观念是，「无法兼顾」是个不好的词，但我却认为它是成功与快乐的关键。 选 3 哲学能帮助你搞定超载的人生，最重要的关键就是有所取舍。 你只专注在每天选择的 3 个面向，就能完全掌握轻重缓急，而且让自己有机会做到最好，这样你会更有干劲，强过好几个星期都注意力涣散。 每天都选不同 3 项，一段时间均摊下来，啊哈，全都兼顾了！

接下来我们来一一简单介绍作者提出的这五个面项：工作、睡眠、家庭、运动、朋友。

工作

你投注你的时间去做，因此而获取报偿、产生价值的计划，收获的形式通常是金钱、热情、意义、对某项伟大事业有所贡献的使命感，或者是完成长期目标的垫脚石。 价值有可能源于一项传统工作、热切的计划、一堂课程或是学校作业、实习工作、公益计划等等，就是投入某些心力而创造出来的产出。

工作是人生最重要的课题之一，但也因为最广泛，广义来说就是你的〝职场生活〞，因为我们通常会花大半辈子以上的时间在投入这一方面，所以若能在工作中又找到〝使命感〞和〝成就感〞，会让我们长期下来更有动力和幸福感。

睡眠

这件麻烦事每天占掉你百分之三十的时间（如果你够好命！ ）

人的时间是固定没办法增加的，所以让自己增加的时间最有效益的方式就是〝提升效率〞，简单来说，做一件事情原本需要 2 小时，你却只花 1 个小时就完成，那这 1 小时就是你赚来的了; 最近我一直在研究如何高效工作，当中过程就发现，想要高效工作，睡得好是一件特别重要的事。

家庭

可能是指你的原生家庭、你创造的家庭，或是你选择的家庭。 不一定与你有血缘关系。 也许你把教会当作你的家庭，也许你有个「前卫」或非传统的家庭。 不管如何定义你生命中的家庭，它就是你的优先选项之一。

这边定义的家庭，也就可以定义你认为人生当中〝最重要〞的人，最普遍的就是父母、兄弟姐姐，甚至有些会有好麻吉之类的，他们对我们的影响非常大，但通常我们却不一定花费很多时间在他们上面。

所以如果透过〝选 3 哲学〞来思考这一件事情，或许短期内事情很忙，很难排出时间，但长期来看、把家庭排入时间，绝对是非常值得的事。

运动

哑铃与汗水，是讲到「运动健身」会马上跳出来的画面，不过对我来说，这个类别的意义比较广，表示照顾自己的健康，包含身体及心理的适应力、好的情绪表达、专注、压力管理，以及有益健康的饮食。

运动对我来说也是非常重要的项目，运动除了可以增强我们的体力和健康之后，对大脑和专注力的提升也是非常有帮助的。

这个项目跟睡眠一样，都是在〝提升工作效率〞上面是非常有帮助的，尤其是脑力工作者更需要，定期运动可以让我们保持专注和大脑的灵活动，做起事情来就事半功倍。 所以在每一周当中，想办法为自己排入「运动」的项目，长期来看是帮忙非常大的。

作者在这一边把运动定义的比较广泛一点，我是觉得可以想成〝健康〞，只要是对〝健康〞有帮助的项目，如：饮食、身心灵… 等等的项目都可以归到这一方面来。

朋友

我个人是把所有好玩的事都归类到这一项。 我们想到的通常是生命中最亲近的朋友，但是这个项目也会让我想到兴趣与嗜好——总之就是工作和家庭之外能让你觉得有乐趣的人或事。

在这边〝朋友〞项目也是定义相对广泛的，可以想成〝娱乐、休闲〞方面的，也可以把〝兴趣〞方面的都算在这一个项目上。

朋友 （ 同事 ） 是我们花最常时间在他们身上的，工作上的同事相处、下班后的朋友聚聚等等，我们总是会愿意花费许多时间在他们身上 （ 通常会随年纪慢慢调低这个比例 ），他通常是我们快乐、幸福的泉源之一。

选3哲学，如何实践

作者把我们的人生归类成五大面向，工作、睡眠、家庭、运动、朋友，我们很难面面俱到，所以想把事情做好，就选三，只能三项。 而且不要浪费一分钟感到罪恶或难过于没选到另外两项，因为，你没选到那些的，明天，或是后天，或是下个月，总是有机会的。

每天都可以从这些选项里挑出新的一组，集中精神在这 3 项，你可以挑和前一天相同的，或是挑不同的 3 项，完全由你决定。

也许是每周固定选某3项，周末是另外三项; 也可以是春天某 3 项，冬天某 3 项; 也许每天都换。 〝选 3 哲学〞让我们短期间能专注投入指定项目中，确保产出的品质，又不会给自己过多的压力，长期又能兼顾所有。 这个策略不失为一个好方法。

也许有人会认为偶尔一两天可以做到5项，但是长期来说，这样真的不利永续。 如果你试图把 5 件事都做好做满，那就是走向「耗竭」。

你没有办法维持在高功率状态把5项都做到最好。 家庭、朋友、工作、睡眠、运动，每天都碰一下，当然做得到。 但是，样样通就样样松，就算达到5项满贯，这也表示每一项都不够享有宁静的深度。

〝选 3 哲学〝能帮助我们搞定超载的人生，最重要的关键就是「有所取舍」。

同时也是你快乐的泉源，当每日做完重要的 3 件事后，所剩下来的时间就是你赚到的，你可以处理任何你感兴趣的事。 你只专注在每天选择的三个面向，就能完全掌握轻重缓急，而且让自己有机会做到最好，这样你会更有干劲，强过好几个星期都体力透支且注意力涣散。 在一天结束前，你知道自己不只做到你订下的 3 项目标，而且每件事都做得很好，这种感觉不是很棒吗？

总结

不平衡发展的人生的确会有牺牲，不过，这种牺牲是良性的，你必须放弃兼顾一切的观念; 你必须愿意说出 「下次吧，健身房，今天不行」，或「这趟进程就不和家人一起了」，甚至「今晚只能睡 4 小时」，还有「今天不回电子邮件」。 总之，工作、睡眠、家庭、运动、朋友，就是不能同时 5 项都选，而且还 5 项都做满做好。

放弃了某件事，或是认清你只是凡人，这种感觉可能会让你很难接受。 但是我发誓，一旦你开始专注，分清楚轻重缓急，选 3 项来做; 一旦你容许自己可以有所取舍，而不是事事兼顾，那么，你会发现自己更开心、更充实，而且在你所选择的事情上更加成功。

1、2023 年搞笑诺贝尔奖（英文）

好奇心是所有科学背后的驱动力，这也许可以解释为什么这么多科学家有时会发现自己在朝着一些明显古怪的研究方向前进。搞笑诺贝尔奖成立于 1991 年，这些奖项旨在表彰那些“先让人发笑，然后让人思考”的研究。

今年 2024 年揭晓了去年获奖的诺贝尔奖，其中一些有意思的项目：

  • 解剖学：罗曼-孔萨里（Roman Khonsari）发现，南半球的人更有可能拥有逆时针方向的发卷，这为人类生物学和遗传学提供了一个奇特的视角。
  • 营养学：宫下荷美（Homei Miyashita）和中村广美（Hiromi Nakamura）探索了电动筷子和吸管如何改变食物的味道。这项研究强调了电如何在不改变实际成分的情况下增强食物的味道。
  • 医学：克里斯蒂娜-范领导的研究小组利用尸体研究人的每个鼻孔是否都有相同数量的毛发。这项研究加深了我们对人体解剖学的了解。
  • 生物学：福特斯-伊利（Fordyce Ely）和威廉-彼得森（William Petersen）证明，奶牛接触到嘈杂的声音和猫时，产奶量会减少，这暗示了压力和环境因素如何影响产奶量。

2、研究人员惊讶地发现：当日托所的空气被净化时，儿童的疾病显着减少（英文）

赫尔辛基的日托中心进行了一项实验，当他们在日托中心安装了高效空气净化器后，孩子们的疾病发生率竟然大幅下降了。具体来说，呼吸道感染减少了 29%，抗生素使用减少了 37%。

更让人惊喜的是，这种改善效果并不局限于呼吸系统疾病。研究发现，空气净化还能降低肠胃道感染的风险。

这些发现主要挑战了日托疾病主要通过飞沫和接触传播的观点。该研究表明，很大一部分感染是通过空气传播的，这意味着空气净化系统可能是减少日托（和其他公共区域）疾病的有效且经济友好的方式。

2、趣味陷阱（英文）

作者讲述了在新泽西州的亚马逊仓库工作时，发现员工们虽然被提供了乒乓球桌、空气曲棍球桌等娱乐设施，但几乎没有人使用这些设施。相反，员工们在高压下工作，休息时间也被严格控制，导致他们无法真正放松或享受工作。

文章通过具体的例子和生动的描述，揭示了亚马逊“努力工作，玩得开心，创造历史”标语背后的虚伪和管理层对员工时间的严格控制。

3、所有地图都错误（英文）

这篇文章深入探讨了GPS和数字地图的工作原理，以及为什么用户在使用 AllTrails 和 Strava 等应用时会遇到轨迹不准确的问题。文章指出，地图是对现实的简化和扭曲，而 GPS 数据也有其误差和局限性。例如，现代智能手机的定位精度在理想条件下为 5 到 10 米，但在实际使用中，受卫星数量、天气、地形等因素影响，误差可能更大。此外，应用程序通常会对原始 GPS 数据进行平滑处理，这可能会修正或引入新的误差。

4、为什么 GitHub 真的赢了（英文）

GitHub 的成功可以归结为两个关键因素：时机和品味。在 GitHub 创立之初，分布式开源版本控制工具刚刚开始被广泛使用，而市场上没有合适的托管平台。GitHub 的创始团队都是产品导向的开源软件开发者，他们关注开发者体验，打破了传统的假设，创造了一个他们自己想要使用的平台。相比之下，其他平台如 SourceForge 和 Google Code 则缺乏这种品味和对开发者需求的理解，最终无法与 GitHub 竞争。

5、抑郁症是什么？（英文）

抑郁症是最常见的心理健康问题之一。据估计，2021 年全球有 4% 的人在那一年经历过抑郁症。

这篇文章探讨了抑郁症的本质及其与情绪和决策的关系。作者指出，抑郁症不仅仅是神经化学的随机紊乱，而是与环境互动的结果。情绪被视为对环境状况的信号，帮助我们做出改变或坚持的决策。抑郁症可能源于我们对现状的高期望，当这些期望无法实现时，情绪低落就会加剧，导致抑郁。文章还引用了心理学家Randolph Nesse和Daniel Nettle的观点，解释了情绪在进化中的作用及其对抑郁症的影响。

6、设计有效的组织（英文）

这篇文章探讨了 Stafford Beer 提出的“生命系统模型”（Viable System Model，VSM），它是为组织设计的一种系统化管理工具，帮助复杂的组织结构在不断变化的环境中保持韧性和生存能力。

VSM 强调每个子系统必须拥有一定程度的自主性，但同时又与更大系统有机协作，从而形成灵活应对外部变化的组织。

文章还提到，传统的科层制和中央集权结构在面对快速变化的现代环境时往往缺乏反应速度，而 VSM 提供了一种更具弹性的组织设计思路。

英国国家电网，以及其在智利国家计划中的成功案例，展示了这一模型在实践中的可行性和有效性。这种管理框架对于科技企业、创业公司等面临复杂动态市场环境的组织尤为重要。

7、理解意识（英文）

作者 Gregg Henriques 博士提出了一个统一知识理论，试图解释意识的本质。他认为，意识不仅仅是大脑的产物，而是一个复杂的信息处理系统。想象一下，如果我们的大脑是一台超级计算机，那么意识就是运行在这台计算机上的终极操作系统！

Henriques 博士的理论横跨了神经科学、心理学、哲学等多个领域。他提出，意识是由多个层面组成的，包括感知、情感、认知等。这就像我们开发的软件系统，也是由多个模块组成的，每个模块负责不同的功能。

最有趣的是，这个理论还试图解释为什么我们会有”自我”的感觉。它认为，自我意识是大脑为了更好地处理信息而产生的一种虚拟接口。这不禁让我联想到我们在开发中常用的 GUI 界面，它让用户更容易理解和操作复杂的底层系统。

对于我们这些技术从业者来说，这个理论提供了一个全新的视角来思考人工智能和机器意识。如果意识真的是一种复杂的信息处理系统，那么未来我们是否有可能创造出具有真正意识的 AI 呢？

8、如何获得无限的「内在能量」？

饭饭的又一宝藏文章。我们本身就拥有无限的热情、精力和心力。只不过我们需要一些科学的方法对待和管理自己的能量和精力。

不要用外在的道理、各种「应该思维」来枷锁住自己，这只会让自己活在牢笼当中。倾听自己每一个想法，认真对待它，满足它，让它信赖你，你就会拥有越来越多直至用不完的能量。

9、《黑神话：悟空》讲了一个什么故事？

游戏中的世界设定继承了《斗战神》的背景，其中神佛的永生并非永恒不变，每过一万两千年后，想要延续生命就必须借助蟠桃。因此，蟠桃园的控制权成为了神佛争夺的核心。而孙悟空的反叛，实际上是对这种权力结构的挑战，因为他不仅拒绝接受现有的秩序，还试图通过自己的方式获得永生，这无疑触动了天庭与灵山的根本利益。

孙悟空辞去佛位，回到了花果山，表面上看起来像是放弃了所有，但实际上是在为自己的重生做准备。他知道自己无法直接对抗整个体系，因此选择了一个巧妙的方法——找到一个能够继承他所有力量的新猴子。这个新猴子，也就是玩家在游戏中扮演的角色，肩负着收集孙悟空六根，实现大圣重生的重任。

好奇星人

1、趴着午睡，是怎么毁掉你身体的 | 答案如下

相信不少人有在办公桌上趴着小憩一会儿的习惯，尤其是在赶项目的时候。但是，你知道吗？这种看似无害的小习惯，可能正在悄悄地伤害你的身体！

首先，趴着睡觉会导致颈椎问题。想象一下，当你趴在桌子上时，脖子是怎样一个扭曲的姿势？长期保持这种姿势，很容易引发颈椎病。作为整天对着电脑的我们来说，颈椎健康尤为重要，可不能因为一时偷懒而埋下健康隐患啊！

其次，趴睡会影响呼吸和血液循环。当你趴着时，胸腔被压迫，肺部无法充分扩张，导致氧气供应不足。这就像是给你的”生命服务器”造成了阻塞，长期如此，可能会引发一系列健康问题。

更要命的是，趴睡还可能导致面部变形！没错，你没听错。长期趴睡会使脸部受压，导致皮肤弹性下降，甚至可能改变面部骨骼结构。

那么，我们应该怎么午睡呢？专家建议，最好的午睡姿势是平躺或侧卧。如果在办公室，可以考虑购置一个小型折叠床或者舒适的办公椅。记住，健康的午睡不仅仅是为了消除疲劳，更是为了提高下午的工作效率。
