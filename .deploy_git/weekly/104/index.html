本周讨论：是什么造就了理想的工作？

（本篇来源：我们回顾了60多项关于梦想工作的研究。以下是我们的发现。）

我们都希望找到一份满意的工作，但是，什么样的工作才能让我们真正满意呢？

有些人认为答案可能是激情或者是简单的高薪，但研究表明，梦想工作的六个要素，不包括收入，也不像“追随你的激情”那么简单。

事实上，追随你的激情会让你误入歧途。史蒂夫乔布斯在进入技术领域之前对禅宗充满热情。玛雅·安杰卢（Maya Angelou）在成为著名诗人和民权活动家之前是一名卡吕普索舞者。

相反，你可以通过做你觉得愉快和有意义的工作来培养激情。关键是要擅长帮助他人的事情。

要找到梦想的工作，请寻找：

  1. 你擅长的工作；
  2. 帮助他人的工作；
  3. 支持性条件：让你进入心流状态的参与性工作，支持同事，没有不公平工资等重大负面影响，以及适合你个人生活的工作。

我们出错的地方

人们试图实现梦想工作的通常方式是想象不同的工作，并考虑它们看起来有多令人满意。或者他们会思考过去感到满足的时光，并自我反思对他们来说最重要的事情。

然而，研究表明，尽管自我反省是有用的，但它作用相当有限。

你可能会想起自己对即将到来的假期或派对充满期待，但当这些事情真的发生时，你发现它们并不如你想象的那么美好。这种现象在过去的几十年的研究中被反复证实：我们并不总是能准确预测什么会让我们感到快乐，而且我们往往低估了自己在这方面的不足。

事实上，我们甚至不能准确地记住不同的体验给我们带来的满足感。一个普遍的误区是，我们常常根据经历的结局来评价整个过程：如果你在一次愉快的假期的最后一天错过了航班，你可能就会觉得这次假期很糟糕。

我们经常通过结局来判断体验的乐趣这一事实，会导致我们做出一些奇怪的选择。

–丹·吉尔伯特教授，《绊倒幸福》

这就意味着，我们不能仅仅依赖自己的直觉来决定哪种工作最适合我们，我们需要一种更系统的方法。

两个被高估的目标，成就了一个充实的职业生涯

人们经常认为梦想的工作是高薪和轻松的。

2015 年，由 CareerCast 提供的美国领先职位排名之一根据以下的标准对工作进行评级：

  1. 报酬高吗？
  2. 将来会得到高薪吗？
  3. 压力大吗？
  4. 工作环境不愉快吗？

基于此，最好的工作是：精算师。也就是说，使用统计数据来衡量和管理风险的人，通常在保险业。

的确，精算师对自己的工作比平均水平更满意，但他们并不是最满意的。只有 36% 的人说他们的工作是有意义的，因此，精算师并不是一个特别充实的职业。

因此，CareerCast 列表并没有捕获所有内容。事实上，有证据表明，金钱和避免压力并不那么重要。

金钱让你更快乐，但只有一点点

“你买不到幸福”是陈词滥调，但与此同时，更高的薪水是人们在寻找新工作时的首要任务。此外，当人们被问及什么最能改善他们的生活质量时，最常见的答案是更多的钱。

但研究发现：金钱确实会让你快乐，但只是一点点。

不要以低压力为目标

许多人想找一份压力不太大的工作。的确，在过去，医生和心理学家认为压力总是不好的。但一些现代压力文献反应，今天，情况有点复杂。

一个难题是，对高级政府和军事领导人的研究发现，尽管睡眠时间更少，管理更多的人，并且有更高的职业要求，但他们的压力荷尔蒙水平较低，焦虑较少。一个得到广泛支持的解释是，拥有更大的控制感（通过设定自己的时间表并确定如何应对他们面临的挑战）可以保护他们免受职位要求的影响。

有一份要求很高的工作是不好的——这很无聊。有超出你能力的要求也是不好的：它们会造成有害的压力。最佳点是对你的要求与你的能力相匹配的地方——这是一个充实的挑战。

与其寻求避免压力，不如寻求支持性的环境和有意义的工作，然后挑战自己。

理想工作六要素

1. 引人入胜的工作

真正重要的不是你的薪水、地位、公司类型等等，而是你每天、每小时做什么。

引人入胜的工作是吸引你，保持你的注意力，给你一种流动感的工作。这就是为什么花一个小时编辑电子表格会感觉像是纯粹的苦差事，而花一个小时玩电脑游戏可能感觉根本没有时间：电脑游戏的设计尽可能吸引人。

有什么不同？为什么电脑游戏引人入胜，而办公室管理员却不吸引人？研究人员确定了四个因素：

  1. 决定如何执行工作的自由。
  2. 明确的任务，明确定义的开始和结束。
  3. 任务类型的多样性。
  4. 反馈，让您知道自己做得有多好。

在一项主要的荟萃分析（r=0.4）中，这些因素中的每一个都被证明与工作满意度相关，专家们普遍认为它们是工作满意度最经经验验证的预测指标。

也就是说，玩电脑游戏并不是充实生活的关键（不仅仅是因为你不会得到报酬）。那是因为您还需要下面这些要素。

2. 帮助他人的工作

但当被问及时，超过四分之三的人表示他们觉得它们没有意义：

  • 收入分析师
  • 时装设计师
  • 电视新闻广播导演

然而，几乎每个从事这些工作的人都认为这些工作是有意义的：

  • 消防员
  • 护士/助产士
  • 神经外科医生

关键的区别在于，第二组工作似乎可以帮助其他人。这就是为什么它们是有意义的，这就是为什么帮助他人是我们的第二个因素。

越来越多的证据表明，帮助他人是生活满意度的关键因素。做志愿者的人不那么沮丧，更健康。一项针对 23 项随机研究的荟萃分析表明，行善会让给予者更快乐。一项全球调查发现，向慈善机构捐款的人对生活的满意度与收入是其两倍的人一样。

帮助他人并不是通往有意义职业的唯一途径，但研究人员普遍认为这是最强大的途径之一。

3. 你擅长的工作

擅长你的工作会给你一种成就感，这是积极心理学发现的生活满意度的关键因素。

它还使您能够就充实工作的其他组成部分进行谈判——例如从事有意义的项目、承担引人入胜的任务和赚取公平报酬的能力。如果人们重视您的贡献，您可以要求这些条件作为回报。

由于这两个原因，技能最终胜过兴趣。即使你热爱艺术，如果你把它当作职业但不擅长，你最终也会为你不关心的公司做无聊的平面设计。

这并不是说你应该只做你已经擅长的工作——但你确实希望有潜力擅长它。

4. 与支持我的同事一起工作

显然，如果你讨厌你的同事，为一个来自地狱的老板工作，你就不会满意。

由于良好的人际关系是拥有充实生活的重要组成部分，因此能够在工作中与至少几个人成为朋友很重要。这可能意味着至少与几个与您相似的人合作。

但是，您不需要与所有人成为朋友，甚至不需要像所有同事一样。研究表明，也许最重要的因素是当你遇到问题时，你是否能从同事那里得到帮助。一项主要的荟萃分析发现，“社会支持”是工作满意度的首要预测因素之一（r=0.56）。

那些不愉快和与你不同的人可以成为给你最有用的反馈的人，只要他们关心你的兴趣。这是因为他们会原封不动地讲述它，并有不同的观点。亚当·格兰特（Adam Grant）教授称这些人为“令人不快的给予者”。

当我们想到梦想的工作时，我们通常会关注这个角色。但与谁一起工作几乎同样重要。一个糟糕的老板会毁掉一个梦想中的职位，而如果和朋友一起完成，即使是无聊的工作也会很有趣。那么在选工作的时候，能和职场上的一些人交朋友吗？更重要的是，工作场所的文化是否使获得帮助、获得反馈和一起工作变得容易？

5. 没有重大负面影响的工作

要满意，以上一切都很重要。但是你也需要没有让工作不愉快的事情。以下所有情况都与工作不满意有关。

  • 通勤时间长，特别是如果乘坐公共汽车超过一个小时。
  • 工作很长的时间。
  • 你觉得不公平的报酬。
  • 工作没有安全感。

虽然这些听起来很明显，但人们经常忽视它们。长途通勤的负面后果可能足以超过许多其他积极因素。

6. 适合余生的工作

你不必从工作中得到充实生活的所有要素。有可能找到一份支付账单并在其他附属项目中表现出色的工作;或通过慈善事业或志愿服务找到意义感;或者在工作之外建立良好的关系。

有著名的例子——爱因斯坦在 1905 年度过了他最富有成效的一年，当时他在一家专利局担任文员。

因此，最后一个因素提醒您考虑您的职业生涯如何与余生相适应。

小结

本篇文章内容已经够长了，原文中还对「你应该追随你的激情吗？」和「做有贡献的事」这两个问题进行了深刻的讨论，如果感兴趣，可以去原文中阅读一下。

总之，要有一份梦寐以求的工作，不要太担心金钱和压力，也不要无休止地自我反省去寻找你真正的激情。

相反，善于做一些帮助他人的事情。这对你最好，对世界也最好。

1、一万小时法则的虚假承诺（英文）

这篇文章对“一万小时定律”进行了深入的探讨和批判。一万小时定律是由马尔科姆·格拉德威尔在其著作《异类》中提出的，他认为任何人只要投入一万小时的刻意练习，就可以在任何领域达到世界级水平。然而，文章指出，这个理论存在许多误解和问题。

首先，原始研究并非关于投入特定时间的学习，而是关于刻意练习，这是一种系统的、有目的的练习，目标是提高表现，需要集中注意力，而不是无意识的重复。

其次，研究的主要研究者并不同意一万小时定律。他们的研究只是表明“最好的”小提琴手比“好的”小提琴手投入了更多的刻意练习时间，这并不能保证任何人只要投入一万小时的刻意练习就能成为任何领域的专家。

最后，普林斯顿的一项研究表明，实践在各个领域的表现中只占12%的差异，这也进一步证明了一万小时定律的问题。

文章还提出了一种新的学习策略——交错学习，即同时练习多个并行技能，这种方法比传统的阻塞学习（先掌握一项技能再学习下一项）更有效。

7、#95: 我以为我懂了

当你觉得自己已经懂了的时候，或许可以问自己几个问题：

  • 我可以用自己的话表述出来吗？
  • 它背后的基本原理是什么？跟我已经知道的有什么联系？
  • 有练习的方法吗？实践后有没有搞清楚它适用在哪些情况，而在哪里不适用？

如果这听起来太累了，提醒自己：困难的路越走越容易，容易的路越走越难；做长期来看正确的事情，不管它困难与否。

1、锻炼后一身汗，能不能马上洗澡？

结论是：能。有汗不能马上洗澡，是没有医学根据的说法。洗澡没那么多讲究，出汗不舒服了，想洗就洗，什么水温舒服就用什么水温洗。

最常见的几种没道理的说法是：

  • 说法一：出汗时毛孔扩张，洗澡时温度下降毛孔收缩病毒细菌入侵，容易导致感冒生病。
  • 说法二：出汗时洗澡，“寒气”会从扩展的毛孔入侵体内导致生病。
  • 说法三：出汗时洗澡毛细血管骤然收缩会导致血液回流增加心脏负担，容易导致猝死。

言论

1、

这个世界上，哪里都没有芭比乐园。虽然现状不容乐观，但某种程度上，这种超越了国家、语言和地理层面的共识也会成为一种更广泛的联结，而这种联结所产生的力量，一定会改变些什么。

–爆火的《芭比》：取悦了女性，却冒犯了男性？

2、

在构建一个新事物时，一个好的第一步是构建一个什么都不做的东西。这样，你至少知道你从一个好地方开始。

很多时候，我看到相对缺乏经验的开发人员潜入并开始编写一个大复杂的东西：然后他们甚至无法编译它，因为它是如此庞大和复杂。

从什么都不做的事情开始。确保你什么都做不了。只有这样，你才应该开始做出改变，这样它才能开始做一些事情。这样，您就知道您遇到的任何问题都与您尝试做某事有关。

–在你尝试做某事之前，确保你什么都不能做（英文）

3、

当过程依赖于比任何参与者都知道的更多的人时，不同的激励措施可能会导致从整体上看结果似乎效率低下。如果组织的参与者没有足够的资源，情况可能会变得更糟。沟通可以提供帮助，但不一定会产生最佳结果。变化加剧了困难。我所描述的内容对于任何大型组织都是通用的。

–为什么在一个大型组织中，一切都如此困难？（英文）

4、

早期的互联网，每个网站都像是进入创作者的思维世界。网页充满了动态背景、光标轨迹和独特的图标，每次浏览都是一次不可预测的冒险。

随着社交媒体巨头和商业利益的崛起，互联网发生了变化。个性化、多彩的网页变得越来越统一，符合现代设计标准。

–互联网变得没有灵魂，我讨厌它（英文）
