本周讨论：为什么大多数人的改变计划都会失败

（本期来源：如何在一天内彻底改变你的人生 | Dan Koe（英文））

这篇文章来自 Dan Koe，一位在 X(Twitter)上拥有超过 50 万关注者的独立创作者。他专注于研究个人发展、创业和生产力，擅长将心理学理论转化为可操作的方法。

2026 年 1 月，他发布了这篇长文，讨论为什么大多数人的新年计划会失败，以及如何真正改变自己。文章篇幅很长，涉及目的论、身份形成机制、心智发展阶段等多个心理学概念，并提供了一套完整的自我反思协议。

以下是对原文的改写。我保留了核心的心理学框架和实践方法，但调整了叙述方式，使其更符合客观分析的风格。

从新年决心说起

每年一月，健身房总会迎来一波新会员潮。这些人信心满满地办卡，计划着要改变身材、提升自律。但数据显示，其中约80%的人会在两个月内放弃。

这不是个例。无论是创业、学习新技能，还是改变生活方式，大多数人的改变计划都会在短时间内夭折。问题不在于缺乏动力或意志力，而在于他们从根本上误解了改变的本质。

改变的真正障碍

你追求的不是行动，而是身份

设定目标时，人们通常会关注两个层面:一是改变行动，二是改变自己。前者是次要的，后者才是关键。

想象一个健美运动员。你觉得他每天需要”努力”才能保持健康饮食吗?事实恰恰相反——对他来说，吃垃圾食品才需要努力。这不是因为他有超人的自律，而是因为健康饮食已经成为他身份的一部分。

关键在于:如果你想要某种生活结果，就必须提前很久拥有那种生活方式，而不是等达成目标后再享受。

这解释了为什么很多人减肥后会反弹。如果你只是忍耐着完成减肥计划，等着”解放”后重新享受生活，那么你从未真正改变自己的身份。一旦目标达成，你会自然回到原来的状态。

你的行为暴露了你的真实目标

心理学中有个概念叫”目的论”:所有行为都是为了达成某个目标。这听起来显而易见，但大多数人不愿意承认。

比如，你说想创业，却一直拖延不开始。表面上看是缺乏自律，实际上你在追求另一个目标——避免失败后的评判。

再比如，你抱怨工作没意义，却年复一年留在那里。你嘴上说想冒险，行为上追求的却是安全感和社会认可。

阿尔弗雷德·阿德勒说过:”只信任运动。生活发生在事件层面，而非言语层面。”你的行为比你的话语更诚实。

身份是如何形成的

理解了目的论，就能理解身份的形成机制:

  1. 你想实现某个目标(无论有意识还是无意识)
  2. 通过这个目标的视角感知现实
  3. 只注意到能帮你实现目标的信息
  4. 采取行动并获得反馈
  5. 不断重复，直到行为变成自动反应
  6. 这种行为成为”你是谁”的一部分
  7. 你会捍卫这个身份，保持心理一致性

问题在于，这个循环从你还是孩子时就开始了。你的父母教你如何生存，而他们自己也继承了上一代的信念系统。除非主动打破这个循环，否则你会一直按照被编程的路径生活:上学、工作、退休，一条已知但未必适合你的道路。

改变需要经历的阶段

发展层次的客观规律

心理学研究发现，人的心智会经历可预测的发展阶段。从马斯洛的需求层次理论，到螺旋动力学，多个模型都指向同一个事实:你的思维方式决定了你能看到什么样的可能性。

以下是简化的9个发展阶段:

  1. 冲动期:感觉和行动没有界限
  2. 自我保护期:学会隐藏真实想法以求安全
  3. 顺从期:完全认同群体规则，无法理解不同选择
  4. 自我觉察期:开始意识到内心与外界的冲突
  5. 尽责期:建立自己的原则体系，为自己负责
  6. 个人主义期:发现原则也受环境影响，开始质疑
  7. 战略家期:在认识到局限的同时与系统合作
  8. 构念意识期:把所有框架视为有用的工具而非真理
  9. 统一期:工作、休息和娱乐融为一体

大多数读到这里的人处于第4到第8阶段之间。好消息是，无论你在哪个阶段，通过下一阶段都有规律可循。

智力的真正定义

Naval Ravikant说:”智力的唯一真正考验是你是否能从生活中得到想要的东西。”

从这个角度看，智力不是IQ分数，而是通过反复试错达成目标的能力。这涉及三个要素:

  • 能动性:你是否愿意行动
  • 机会:环境是否提供可能性
  • 智力:你能否利用机会

控制论给出了智能系统的特征:有目标、采取行动、感知结果、对比反馈、再次调整。飞机偏离航线后会自动纠正，恒温器感应温度变化后会启动——这就是智能。

低智能的标志是无法从错误中学习，遇到问题就放弃。高智能意味着相信任何问题都能在足够长的时间内解决，并持续迭代直到找到答案。

一天内重置生活的方法

为什么需要深度质询

很少有人会真正质询自己的生活。你可以从一个人说话的方式看出来——他们是在重复听来的观点，还是经过深思熟虑。

改变需要的不是更多信息，而是更深的自我认识。以下是一个全面的自我质询方案，需要投入一整天时间完成。

早晨:心理挖掘

预留15-30分钟，诚实回答这些问题:

揭示当前痛苦:

  1. 你学会与之共存的那种沉闷不满是什么?
  2. 你反复抱怨什么，却从不真正改变?
  3. 对每个抱怨:观察你行为的人会认为你真正想要什么?
  4. 你生活中的什么真相，让你无法向尊敬的人坦白?

创造反向愿景:

如实回答这些问题后，你会感到不安。这种不安很重要——它是改变的燃料。

  1. 如果未来五年完全不变，描述一个普通的周二:你在哪里醒来?身体感觉如何?早上到晚上都做什么?
  2. 坚持十年会怎样?你错过了什么机会?谁会放弃你?
  3. 人生尽头回望:你从未让自己尝试或成为什么?
  4. 你生活中有谁正在经历这样的未来?想到成为他们，你有什么感受?
  5. 你必须放弃什么身份才能改变?付出什么社交代价?
  6. 你最尴尬承认的原因是什么?
  7. 如果现在的行为是自我保护，你在保护什么?代价是什么?

建立最小可行愿景:

  1. 三年后，你真正想要的生活是什么样的?一个普通周二的细节?
  2. 你需要相信自己什么，那种生活才会感觉自然?
  3. 如果你已经是那个人，这周会做什么?

白天:打断自动模式

在手机设置随机提醒，包含以下问题:

  • 上午11:00:我现在在避免什么?
  • 下午1:30:过去两小时的行为说明我想要什么?
  • 下午3:15:我在走向讨厌的生活还是想要的生活?
  • 下午5:00:我假装不重要的最重要的事是什么?
  • 晚上7:30:我今天做什么是出于身份保护而非真实欲望?
  • 晚上9:00:今天什么时候最有活力?什么时候最死气沉沉?

通勤或休息时思考:

  • 如果不再需要别人认为我是某种人，会有什么变化?
  • 我在哪里用生命换取安全?
  • 我想成为的人中，明天可能成为的最小版本是什么?

晚上:综合洞见

经历一天的思考后，回答:

  1. 你觉得自己被困住的最真实原因是什么?
  2. 真正的敌人是什么?(不是环境或他人，而是内在模式)
  3. 用一句话表达你拒绝让生活变成的样子
  4. 用一句话表达你正在构建的目标

设定目标作为视角:

目标不是终点，而是引导行动的透镜。

  1. 一年视角:什么事发生了，你就知道打破了旧模式?
  2. 一月视角:什么必须发生，才能让一年目标成为可能?
  3. 每日视角:明天你能做的2-3个行动是什么?

把生活变成游戏

米哈伊·契克森米哈伊在研究心流状态时发现，最佳体验发生在目标明确、技能与挑战匹配的时刻。游戏之所以让人沉浸，是因为它具备这些要素。

你可以用同样的结构设计生活:

  • 反向愿景:如果失败会怎样?(游戏中的死亡)
  • 愿景:你想达成什么?(游戏中的胜利条件)
  • 一年目标:唯一优先级(游戏中的主线任务)
  • 一月项目:获取经验和战利品(游戏中的Boss战)
  • 每日任务:推进进程的具体行动(游戏中的支线)
  • 约束:你不愿意牺牲什么?(游戏规则)

这些要素像同心圆一样保护你的注意力，屏蔽干扰。随着时间推移，这种专注会变成你的自然状态。

最后的提醒

这套方法不保证对所有人都有效。它需要你处于准备改变的阶段，需要你诚实面对自己。

但如果你真的投入一天时间完成这个过程，至少会获得一个深刻洞见，足以改变你的轨迹。

改变不是变得更自律，而是变成另一个人——一个自然而然做出不同选择的人。

1、AI 时代生存指南（英文）

文章较长，但核心观点是：当硅基智能接管所有必要性劳动，碳基生命的唯一出路是成为”意义架构师“——不是与机器竞争效率，而是提供人类彼此需要的真实体验。未来属于有品味、有能动性、有独特视角的创作者，因为 AGI 可以生成内容，但无法生成你的灵魂。

2、人类最高成就获得机制的最新研究（英文）

这项涵盖 34,839 名顶尖人才（诺奖得主、奥运冠军、顶级棋手、音乐家）的 Science 综述揭示了三个反直觉真相：

  1. 青少年时期的顶尖表现者与成年后的世界级成就者，是两组几乎不重叠的人群（重合度仅约10%）；
  2. 大多数世界级成就者在早期表现平平，其巅峰表现与早期表现呈负相关；
  3. 两类人的成长路径截然相反：青少年精英通常通过早期高强度专精训练快速成功，而世界级大师则往往起步较晚，早期涉猎广泛（多学科/多项目），进步更为渐进。

4、不要陷入反AI的宣传（英文）

他亲证 Claude Code 能在几小时内完成过去需要几周的任务——从 UTF-8 行编辑库到 BERT 推理引擎纯 C 实现，甚至重构 Redis Streams。

5、我们花了两天时间，终于造出了能自我进化的Skills管理器。

卡神的最新分享。核心方案是：

  1. github-to-skills为GitHub项目生成的技能注入身份ID（仓库地址和版本哈希）；
  2. skill-manager实现技能状态检查、更新和删除；
