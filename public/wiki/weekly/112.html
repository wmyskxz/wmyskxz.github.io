本周讨论：社会权力的游戏

（本期来源：富于理性#94 社会权力的游戏）

1959 年，学界提出权力的六种基础（bases of social power）。其中，「社会权力」（social power）指影响他人行为，并抵制他人影响自己的能力。

虽然「权力基础模型」不能提供对权力逻辑最深度、全面的解析，但它确实是一个简洁、好上手的模型。

使用这个模型，我们可以逐一分析以下的几个问题：

  • 作为最能赚钱的玩家，巴菲特如何参与社会权力的游戏？
  • 拿破仑如何崛起，为何摔落？
  • 82 年生的金智英，为何会精神崩溃？
  • 当红流量明星，面临着什么隐忧？
  • 刚出新手村，年轻人如何在社会上立足？

这些问题看似千差万别，但有一个简洁的思维模型，可以帮助思考：权力的基础。

巴菲特的权利密码

以巴菲特为例，我们逐一分析他拥有的各项权力。你会发现，他是一位「六边形战士」：

1）专家权力

首先，他拥有高超的投资技艺，以此获得专家权力。他在近80年的职业生涯中，不断磨练技艺，形成了强大的专业能力，这是他的立身之本，也是其他各项权力之源。

2）奖赏权力与强迫权力

通过积累大量财富，巴菲特拥有了可观的奖赏与强迫权力。

对于旗下公司，巴菲特有权提拔或裁撤公司高管。

他的教导也能帮学生更强地掌握投资的秘诀，比如，对于知名的学生（如李录），他的指导正是一种奖赏。学生的成功，又反过来能增强他的信息权力。

3）合理权力（符合社会规范）

一旦成家立业，成为公司掌门人、一家之长，巴菲特可以自如地行使相关权威，施行合理权力。

他喜欢在股东信里强调伯克希尔·哈撒韦交了多少税，这也在巩固自己的合理权力。

4）参照权力（因为他人对你有尊重与爱，能施加影响力）

在「门徒」眼中，巴菲特是奥玛哈的圣人。

最重要的，当然是他把几乎全部家产捐回社会，生活低调简朴，这是跨文化普遍被认可的美德。

此外，他各方面的细节也做得很好。

就社会义务而言，90 年代末，他协助收拾长期资本公司倒台后的烂摊子；2008年，在金融危机风雨飘摇时，他出来喊话“Buy America, I am”，为金融界注入信心。

生涯中晚期，他积极布道，分享关于人生和投资的智慧。

就与人交往而言，他精心研究过表达的艺术，擅长自嘲，风趣幽默。例如，他在1998年的一次演讲中，仅仅80分钟就让听众大笑 20 次。

由此，他得到多方的尊重与爱戴。这也是幸福的源泉。

5）信息权力

想象一下，如果有一位能干的、清贫的年轻人，他能获得巴菲特当前掌控的信息，哪怕只有1%，也足够他成为亿万富豪。

以上这些权力相辅相成。在社会权力的游戏中，巴菲特绝对是技艺最高超的玩家之一。

拿破仑的崛起与衰落

拿破仑则提供了典型案例。他起初获取权力，然后善用权力，最终滥用权力。

年轻时，拿破仑原本是一位贫穷军官，法国大革命期间，他凭借军事才华脱颖而出，立下赫赫军功，这是专家权力。

随后十年，他带领军队从一个胜利走向另一个胜利，步步高升，这使他得以掌握更强的武力，即强迫权力。

他借武力改组政府，成为最高执政之一，有力维护社会秩序，获得合理权力。

他作为击退外敌的民族英雄，作为广受爱戴的凯旋将军，得到巨大的参照权力。

最终，他在 35 岁时自立为帝，获得法国最高的奖赏与强迫权力。

称帝之后，拿破仑权力日益昌隆。对外，他仍然屡战屡胜，成为欧洲的「王中之王」。对内，他推动编撰《拿破仑法典》，建立法律秩序，促进社会的稳定与发展，也巩固了自身权力。

然而，随着权力欲望一再膨胀，失败的种子也已播下。

为了惩戒英国，他下令欧洲大陆诸国不得与英国通商，此时，他的合理权力就已经受到损害。因为当时英国主宰海上贸易，中断贸易后，受苦的是大陆上的万千民众，比如砂糖很快就出现了短缺。当俄国违抗他的禁令时，拿破仑怒火中烧，远征俄国，大败而归，最终发展至滑铁卢的溃败。武力（强迫权力）随着胜利到来，也随着溃败而逝去。

这是他滥用权力的后果，最终也促成他的衰落。

金智英为何发疯

唐纳德·特朗普的前妻伊万娜·特朗普曾有一句名言，“Don’t get mad, get everything.” 一种翻译是「别抓狂，抓牢所有东西。」

地球另一端，比伊万娜·特朗普年轻 33 岁的韩国女性金智英，没有止步于抓狂。作为小说人物，她最终还是「发了疯」。

怎么发疯呢？金智英像是被母亲附身，在与丈夫的父母聚餐时，她会用母亲的语气，说些「大逆不道」的话：「每次过节，你们家规矩太多，都让我家智英准备得太辛苦。过完节，她都全身酸痛。」「亲家公，恕我冒昧：只有你们家人团聚很重要吗？我们也是除了过节以外，没有别的机会可以聚在一起好好看看三个孩子。既然你们的女儿可以回娘家，那也应该让我们的女儿回来才对吧！」

好端端的，为什么她会发疯？

要考究这个问题，请想象两位分别生活在韩国和美国的全职妈妈。

有些劣势是相通的。暂且先不谈自我实现，谈些更接地气的。例如，经济资源有限（缺乏奖赏权力）、信息获取不足（缺乏信息权力），以及需要承担沉重的家务负担（社会赋予男性的合理权力）。

就参照权力而言，在美国等国家，为家庭事务付出的劳动，能得到更充分的尊重与承认（虽然也十分有限）。而在东亚，至少截止目前，所有这类再生产劳动都被严重贬低，因此，如果一位女性花大量精力从事这些工作，她很容易被歧视。

金智英情绪崩溃，开始「被附身」，直接导火索，正是她听到有人称呼自己为「妈虫」。她不由得控诉，「我赌上自己的性命把孩子生下来，甚至放弃了自己所有的生活、工作、梦想，只为了带孩子，我却成了他们口中的一只虫，你说我接下来该怎么办？」

那么，发疯意味着什么？

发疯是跳脱框架，是对压顶式的、不合理的「合理权力」的反抗，是一种变形的强迫权力。虽然，在这个阶段，金智英所给出的惩罚，最终也不过是「非暴力不合作」，说出一些（不中听的）真话罢了。

流量明星的权力

当红男性流量明星是非常有趣的一个群体。

他们主要拥有参照权力，来自粉丝对偶像的爱慕、向往和理想化。娱乐产业极强的造梦能力，与大众的理想化需求、钟情幻想一拍即合，形成明星的权力基础。

因此，对于流量明星，最糟糕的事，就是出现恶性新闻，而且得是精准打击粉丝幻想的恶性新闻。例如，某位年轻演员当年危险驾驶，够成违法行为，但粉丝群体不离不弃，全力帮助他重塑良好形象；而当嫖娼新闻爆出，粉丝群体的幻想直接被粉碎，只留下一地鸡毛。

这些事件，俗称「塌房」。实际上，坍塌的是偶像的参照权力。

经典游戏攻略

刚出新手村，想要在社会上立足的年轻人，可以参考这套经典的游戏攻略：

最靠谱的，是磨练出一门技艺，赢得专业权力；

以此为基础，多为集体、为他人服务，对贡献抱有激情，在组织中担任职务、或赢得权威，得到合理权力；

端端正正做人，真诚地与人交往，从想得到友爱和尊重的人那里得到友爱和尊重，得到参照权力；

在工作与广泛交往中，得到有价值的信息，这是信息权力；

以上四者，可以帮助年轻玩家在社会上立足。

能实现这些的，是非常幸运的玩家。那么，自然应该合理分配积攒的资源，明智地实行奖赏权力与强迫权力，以参与塑造一个更值得期待的未来。

热点新闻

1、狗什么时候成为我们最好的朋友？（英文）

在意大利第二长河——阿迪杰河畔的维罗纳，考古学家们发现了一个铁器时代（2000 年历史）的墓地。其中一个名为 B19 的墓穴里，有一个婴儿和一只狗的完整骨骼，静静地躺在一起。

通过对骨骼的同位素分析，科学家发现这只狗的饮食与众不同，主要是碳水化合物，而且它还有一条曾经折断过的腿。这些细节让人猜测，它可能是家中的宠物，受到了与人类成员相同的关爱。

这个发现再次证明了狗和人类之间深远而特殊的关系。从数万年前古老的狼逐渐演化成为狗，它们不仅是我们的工作伙伴，更是我们情感上的依靠。

2、喷墨打印机不仅仅用于打印（英文）

喷墨打印技术自 20 世纪 80 年代初以来，从嘈杂的办公室设备演变成今天家庭和办公室中不可或缺的静音打印工具。现代喷墨打印头能够在每秒数万次的频率下，精确控制仅有 14 微米直径的墨滴，展现了微电子机械系统（MEMS）技术的精妙。

但喷墨技术的应用已远远超出了打印纸张，涉及基因组学、电路板制造，甚至是 3D 打印等多个领域。

一个例子是将 3D 喷墨打印用于个性化医疗。这个想法是生产为特定患者定制的药物片剂。这种个性化药丸可以包括对个体剂量的简单微调，以及通过修改片剂的结合剂和结构来调整药物的释放速率——从非常快到缓慢和持续。患者可以每天服用一种复方药，而不是每天在复杂的时间表上处理多种药物，这是一种包含多种药物的3D打印药片，每种药物的释放速率都不同。

喷墨系统也特别适用于以薄膜形式打印药物，例如应用于皮肤的透皮贴剂和用于脸颊的颊膜，药物可以直接进入血液，而无需首先通过消化系统。

3、如何让老的免疫系统恢复年轻（英文）

在最新的科学突破中，研究人员发现了一种方法，可以使老年小鼠的免疫系统恢复年轻活力。这项研究不仅为抗衰老领域带来了新的希望，也可能对未来的人类健康产生深远影响。

  • 免疫系统重启：通过靶向老年小鼠的造血干细胞，科学家成功降低了与炎症相关的细胞类型，从而减少了过度炎症并恢复了免疫反应。
  • 科学的进步：这项发表在《自然》杂志上的研究，展示了通过抗体治疗，老年小鼠对疫苗接种的反应更强，对病毒感染的抵抗力也得到了提升。
  • 未来的希望：尽管这种方法还需要数年时间才能在人体中进行测试，但它已经证明了通过调整干细胞种群平衡，可以有效地逆转免疫系统的衰老。

1、当你做 git push 时，GitLab 上会发生什么？（英文）

这篇文章解释了 Githe 和 GitLab 的幕后工作。以下是提炼的一些重点，感兴趣的可以看看：

  • .git 文件夹的秘密：每个使用 Git 的项目都有一个隐藏的 .git 文件夹，它保存了所有 Git 的信息。这次我们重点关注其中的 heads、remotes 和 tags 部分，它们是如何在你推送代码时发挥作用的。
  • 服务器的裸仓库：服务器端的 Git 仅在 .git 文件夹中存储信息，这被称为裸仓库。当你克隆或推送代码时，GitLab 是如何处理这些信息的？
  • SSH 与 HTTP(S) 的对话：无论是通过 SSH 还是 HTTP(S)，GitLab 都有一套独特的机制来处理你的 git push。GitLab Shell 的 gitlab-sshd 和 Workhorse 是如何确保安全和效率的？
  • Gitaly 的高效管理：Gitaly 是 GitLab 的心脏，负责管理 Git 仓库并确保操作的高效性。它是如何通过最小化磁盘 IO 操作来提升性能的？

其中有意思的是，Gitaly 的名字：这是将“Git”与人口荒凉的俄罗斯村庄“Aly”混合在一起，旨在实现最少的磁盘 IO 操作，就像 Aly 的人口最少一样。这种俏皮的命名强调了开发人员的效率和极简主义目标。

2、不要修复任何东西——而是培养接受度：游戏中的错误已经成为功能（英文）

这是一个有趣的话题：游戏开发中的错误如何成为创新的催化剂？这篇文章通过一系列著名游戏的例子，展示了开发者如何将错误转化为游戏特色：

  • 意外的美味：在《矮人要塞》中，玩家竟然可以用酒精来烹饪，这个由错误产生的特性如此受欢迎，以至于开发者决定保留它。
  • 意外的伴侣：《星露谷物语》中，美丽的蓝发女孩阿比盖尔竟然会吃掉你送她的钻石，这个有趣的bug最终成为了游戏中一个独特的亮点。

错误是任何过程的一部分，尤其是在编程中。

这句话提醒我们，即使是在技术严谨的游戏开发中，错误也可以成为创新的起点。让我们拥抱错误，从中学习，不断进步。祝大家编程愉快，游戏愉快！

3、你可以有两件大事，但不能有三件（英文）

在生活中，我们不可能“拥有一切”。实际上，我们只能在三件大事中选择两件来专注。这篇文章提出了一个有趣的时间管理理论，让我们一起来看看。

  • 时间分配：我们每天有24小时，如果按照1950年代的标准来分配，8小时工作，8小时家庭和通勤，还有8小时睡眠和个人清洁。这意味着“工作”和“家庭”是你每周可以投入40多小时的两件大事。
  • 选择的代价：如果你想在全职工作的同时创业，那么你可能会牺牲家庭时间。这也是为什么在成家之前创业会更容易一些。
  • 生活的规则：生活的规则是你可以在生活中拥有两件“大事”，但不可能三件。这些大事包括工作、家庭、社交生活、孩子和配偶等。

文章中还提到了一些发人深思的观点，比如“尝试拥有一切的人，实际上从未真正成功”。它提醒我们，我们需要做出选择，并且接受我们无法同时在多个领域都做到最好。

5、空想特摄的浪漫：奥特曼入坑指南

探索奥特曼系列的历史，这个几乎成为文化符号的系列，不仅仅是儿童的英雄故事，它还承载着深刻的社会意义和世代的记忆。以下是本文的精华摘要：

  • 空想特摄的起源：奥特曼，或称Ultraman，诞生于60年代的日本，一个战后反战情绪高涨却又渴望战斗场面的社会。在不能展现国与国战斗的限制下，怪兽电影成为了一种新的表达方式。
  • 昭和到令和：奥特曼经历了昭和、平成到令和三个时代，每个时代的奥特曼都有其独特风格，从初代奥特曼的英雄梦想到平成三杰的科幻冒险，再到令和时代的新生代奥特曼，每一代都有其标志性的变化。
  • 社会影响力：奥特曼不仅是儿童的偶像，也是成年人的怀旧之物。它通过屏幕上的英雄形象，鼓励着全世界的孩子们在逆境中坚持，并成为了一个时代的文化标志。

“奥特曼就是空想的浪漫，让我们对‘英雄’这个词有了一个具象化的认知。”这句话深刻地表达了奥特曼对多代人心中英雄概念的塑造和影响。它不仅是一部特摄剧，更是一种文化现象，激发了无数人心中对正义和勇气的向往。🌟

  • 速溶咖啡的进化：速溶咖啡不再单调，喷雾干燥和冻干技术的对比，让你选择适合自己口味的咖啡。
  • 即饮咖啡的便捷：即饮咖啡省去了繁琐步骤，直接享受咖啡的快感，尤其是黑咖啡，简单纯粹。
  • 挂耳咖啡的魅力：挂耳咖啡提供了接近手冲体验的便利，多样的选择让每一杯都充满期待。

7、亲密关系的牢笼

世人瞩目的“谷歌中国工程师杀妻案”目前还在审理，虽然已有不少媒体跟进，但整个案子仍笼罩在重重迷雾之中，人们最感到困惑不解的是：像这样优秀的年轻人为什么要自毁前途？按说他们生活如此美满，到底哪里出了问题，怎么会走到这一步？杀人动机又是什么？

在这篇深刻的文章中，作者维舟探讨了亲密关系中的控制欲如何变成了一种无形的束缚，甚至可能导致悲剧的发生。以下是文章的几个关键观点：

  • 控制欲的极致：对于某些人来说，控制欲如此强烈，以至于他们宁愿毁掉一切也不愿放手。
  • 社会与文化的影响：文章指出，我们的社会文化常常将这种窒息的控制欲误解为“爱”，而忽视了其负面的压迫性。
  • 自由与独立的重要性：真正的爱应该是支持和尊重个体的自由和独立，而不是将对方囚禁在一种共生的依赖关系中。

“爱不是牢笼。真正的爱必将使人自由。”这些话语强调了个体自由的重要性，并挑战了我们对于爱和关系的传统看法。这篇文章是对那些在亲密关系中感到被束缚的人的一种启示，提醒我们要重新审视我们对爱的理解和期望。

8、S2E7 内容创作者的窘境

这是一期小宇宙的播客类节目（时长 90 分钟）。本来是想放在一周讨论的，但是内容实在是繁杂庞大。所以在这里做一个简单的介绍。是一期很有共鸣的故事。

Jack 的故事是一位内容创作者在现代互联网时代的缩影，反映了许多创业者所面临的挑战和窘境。他的经历展示了在追求艺术创作激情与经济利益之间的艰难平衡，以及在公众评价和个人信念之间的斗争。

Jack 出身于一个经济条件良好的家庭，他的父亲是一名流行病学家。早年间，他就展现出对音乐的浓厚兴趣，并拥有了一间装备齐全的音乐工作室。他在斯坦福大学完成了音乐学位，虽然他最初是从物理系转过来的。毕业后，他并没有像许多同学那样进入硅谷的创业圈，而是选择留在家中，专注于创作和上传 YouTube 视频。

Jack 的创作路上，首先面临的是经济回报与投入不成比例的窘境。他在 YouTube 上的视频创作虽富有创意和热情，却很难获得足够的经济收益。他的视频虽然制作精良，却因不符合主流市场口味而播放量有限。他的独特风格，如强调“所见即所听”，禁止对嘴型和假弹，虽然得到了小部分人的认可，但在广阔的网络世界中显得边缘化，难以吸引大众关注。

随着时间的推移，Jack 逐渐感受到了公众误解和评价的压力。尽管他努力保持自己的艺术原则和创作风格，但观众往往更关注娱乐性而非艺术价值，这让他陷入了认同和价值的危机。他的创作被一些人视为寻求关注和流量的手段，而非真正的艺术表达。

作为一个内容创作者，Jack 还面临着职业身份的认同问题。在互联网上，内容创作者往往被边缘化，不被视为传统意义上的职业。这种职业的不确定性和社会的不认可加剧了 Jack 的内心挣扎。他需要不断地证明自己的工作不仅是一种爱好或暂时的追求，而是一种正当的、可持续的职业选择。

与此同时，Jack 也体会到了与传统产业的冲突。他的创作方式与传统音乐产业的运作模式有着明显的差异，这导致他在尝试与音乐产业接轨时遇到了阻碍。他的音乐风格和创作理念难以被传统音乐市场接受，这使得他难以通过传统的音乐产业获得成功。

就在 Jack 几乎要放弃的时候，一个转机出现了。他开始思考，是否有一种方式，可以不依赖传统的音乐产业体系，直接与喜欢他音乐的人建立连接。这个思考最终促成了 Patreon 平台的诞生。Jack 与一个老友共同创立了这个平台，旨在帮助像他一样的独立创作者，通过粉丝的支持来实现创作和生活的自由。

Patreon 的成功，对 Jack 来说是一种救赎。他终于找到了一种方式，既可以坚持自己的音乐理念，又能获得经济上的独立。这个平台不仅改变了Jack的生活轨迹，也为全球的内容创作者提供了新的可能性。

通过 Jack 的经历，我们可以看到互联网时代创业者的多重挑战：如何在追求艺术真实性和商业成功之间找到平衡点，如何在遭受公众误解和评价时保持自我，如何在职业选择上获得社会的认可，以及如何在传统产业和新兴媒介之间寻找自己的位置。
